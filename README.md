**CLI.do** é um projeto desenvolvido em rust, com o intuito de ser uma CLI de gerenciamento e processamento de tarefas dentro de sistemas pessoais. Com isso, foi desenvolvida em rust, com auxílio do pacote crossterm, e para armazenamento de dados, foi usado o banco de dados **SQLite**, de forma que a perssitência de dados fique toda dentro de um único arquivo, de forma a não ser acessado e modificado pelos usuários diretamente, promovendo uma melhor segurança, se comparado a usar um arquivo de texto
